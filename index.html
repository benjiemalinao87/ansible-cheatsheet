Ad-hoc command¶

ansible Host pattern -m <module> [options]

e.g.

$ ansible all -m ping –ask-pass

Shortcut to run a command:

$ ansible all -a “/bin/echo hello”

options: see output of “ansible –help” for now

See ad-hoc commands.
